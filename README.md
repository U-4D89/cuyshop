# Cuyshop
  Cart shopping made with React 
---
 
 
### Pre-requirements 📝
- Node JS


### Instalation 🔧
 - project in your machine
    - ```git clone https://github.com/U-4D89/cuyshop```
 - dependencies
    - ```npm i```
    or 
    - ```npm install```


### Built it with 🔨
- React
- Bootstrap 5
- Sweet Alert


### How to run? 🚗🌫
 ```npm start```
